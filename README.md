#  Galería

## Descripción
- Se pueden filtrar las imagenes por categoría.
- Se abre una ventana modal cuando se hace click sobre una imagen y al pasar el mause por encima de la imagen aparece una descripción.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- Bootstrap
